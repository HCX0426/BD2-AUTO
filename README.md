# BD2-AUTO

一个功能强大的BD2游戏自动化工具，基于Python和PyQt6开发，提供直观的GUI界面和多种自动化任务支持。

## 功能特点

- 🎮 **PC端自动化**：专为PC端BD2游戏设计，完美适配游戏界面
- 🎯 **丰富任务支持**：包含登录、日常任务、副本扫荡、PVP挑战、公会任务、餐厅经营、地图收集等多种自动化任务
- 🖥️ **直观GUI界面**：现代化的图形界面，操作简单易用，支持任务配置和实时状态监控
- 🔍 **智能图像识别**：基于EasyOCR的高精度图像识别技术，准确可靠地识别游戏界面元素
- ⚡ **高效稳定**：优化的自动化执行逻辑，运行稳定高效，资源占用低
- 📱 **可扩展架构**：模块化设计，支持轻松添加新的自动化任务
- 🔧 **灵活配置**：丰富的配置选项，支持根据个人需求自定义自动化流程

## 下载与安装

### 从GitHub Release下载（推荐）

1. 访问项目的[Releases页面](https://github.com/yourusername/BD2-AUTO/releases)
2. 下载最新版本的`BD2-AUTO.zip`文件
3. 解压到任意目录（确保路径中不包含中文）
4. 双击`BD2-AUTO.exe`即可运行

### 从源代码构建

如果您想从源代码构建项目，请参考[DEVELOPMENT.md](DEVELOPMENT.md)文件的开发环境搭建部分。

### 系统要求

- **操作系统**：Windows 10/11（64位）
- **内存**：至少4GB RAM（推荐8GB）
- **硬盘空间**：至少1GB可用空间（包含OCR模型）
- **游戏分辨率**：1920×1080（推荐，其他分辨率可能影响识别准确性）
- **网络连接**：首次运行需要网络连接下载OCR模型

## 使用说明

### 首次运行

1. 解压下载的压缩包
2. 双击`BD2-AUTO.exe`运行程序
3. 首次运行会自动下载OCR模型文件（约500MB）
4. 等待模型下载完成后，程序会自动启动

### 基本操作

1. **选择任务**：在左侧任务列表中选择要执行的任务
2. **配置参数**：根据任务需求调整右侧的配置参数
3. **启动任务**：点击"开始"按钮执行任务
4. **暂停/停止**：任务执行过程中可以暂停或停止

### 任务说明

### 1. 登录任务
- 自动识别游戏登录界面
- 自动点击"开始游戏"按钮
- 支持自动处理登录验证（如果需要）

### 2. 每日任务
- 自动完成每日签到
- 自动领取每日奖励
- 自动完成每日活动任务

### 3. 副本扫荡
- 支持多章节副本自动扫荡
- 自动选择难度和次数
- 自动处理战斗结算

### 4. PVP挑战
- 自动进入竞技场
- 自动选择对手并挑战
- 自动处理战斗结果

### 5. 公会任务
- 自动进入公会界面
- 自动完成公会任务
- 自动领取公会奖励

### 6. 餐厅经营
- 自动经营餐厅
- 自动升级餐厅设施
- 自动收取餐厅收益

### 7. 地图收集
- 自动收集地图上的资源
- 自动处理资源收集完成后的流程
- 支持多地图资源收集

### 8. 其他任务
- 通行证奖励领取
- 强化分解
- 幸运抽奖
- 活动关卡挑战

## 注意事项

1. **游戏窗口要求**：确保游戏窗口在前台且未被最小化
2. **分辨率要求**：推荐使用1920×1080分辨率，其他分辨率可能影响识别准确性
3. **游戏状态**：执行任务前确保游戏处于主界面
4. **权限问题**：建议以管理员身份运行程序
5. **网络连接**：首次运行需要网络连接下载OCR模型

## 常见问题

### Q: 程序启动后无响应
A: 首次运行时会自动下载OCR模型文件（约500MB），请耐心等待。如果长时间无响应，可以尝试关闭程序后重新运行。

### Q: 任务执行过程中出错
A: 请检查以下几点：
1. 游戏窗口是否在前台且未被最小化
2. 游戏分辨率是否设置为1920×1080
3. 游戏是否处于主界面
4. 程序是否以管理员身份运行

### Q: 无法识别游戏界面
A: 可能的原因：
1. 游戏窗口不在前台
2. 游戏分辨率不符合要求（1920×1080推荐）
3. 游戏界面元素发生变化
4. OCR模型未正确下载

### Q: 如何更新程序
A: 从GitHub Release页面下载最新版本的压缩包，解压后覆盖原有文件即可。

### Q: 程序是否支持其他分辨率
A: 目前主要支持1920×1080分辨率，其他分辨率可能会影响识别准确性。

### Q: 如何自定义任务
A: 在程序界面中，选择任务后可以调整右侧的配置参数来自定义任务执行流程。
